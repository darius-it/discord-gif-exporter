Discord GIF Exporter

Easily export your GIFs from Discord, allowing you to use them in other apps. This is also a useful tool to save your GIFs before the Tenor API shuts down.

1. Export JSON from Discord

To use this tool, you need a JSON which lists all of your favorited GIFs from Discord. There are multiple ways to export this:

Using Tampermonkey script You can use the Tampermonkey script which allows you to open Discord in your browser and grab the JSON with one click. If you already have Tampermonkey installed, click here to install the userscript.
Manual method If you don't want to install another browser extension and you know how to use your browser's developer tools, you can use the original snippet that the Tampermonkey extension was derived from: Exporter by Davr1

2. Upload JSON

Use the button below to upload the JSON you have just acquired.

3. Download & Convert

If you have uploaded the JSON above, you are now ready to download all GIFs! By default, the links saved in the JSON point to .mp4 files. If you wish to have actual .gif files, you can select the conversion below. This conversion is done locally using a WASM module.
