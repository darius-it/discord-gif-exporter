# discord-gif-exporter
Small utility to export Discord favorite GIFs from the Browser (using a Tampermonkey Userscript).

# How to use:
1. Install Tampermonkey
2. Open the script and press install
3. Open Discord in the browser, go to any chat and open the GIF picker
4. There should be a button saying "Export GIFs", which will download a JSON file with all your favorited GIFs.

Enjoy! From this point on, you can use the JSON file to download the GIFs for archival purposes.
